three.js分享

three.js简介

  1. WEBGL

        WebGL（Web Graphics Library ）是一种用于呈现交互式2D和3D图形的JavaScript API，WebGL 可以为 HTML5 Canvas 提供硬件 3D 加速渲染，这样开发人员就可以借助系统显卡来在浏览器里更流畅地展示 3D 场景和模型。


  2. ThreeJS

        three.js是一个javascript的开源三维引擎。three.js对WebGL以面向对象方式进行了封装，使得开发人员可以不用再关注底层的3D图形渲染的细节，利用three.js提供的API就可以很容易的创建出各种三维场景。

https://threejs.org

three.js基础组件

     场景，相机，渲染器，几何对象，模型，材料，光源，动画以及辅助工具等等

  1. 相机（Camera）

         相机类似于我们的眼睛，它的作用是定义可视域，即确定哪些图形元素是可见的


  2. 场景(Scene)

         场景类似于我们所处世界，相当于一个容器，可以用来容纳各种图形元素,图形元素只有添加到场景中，其坐标、大小等才有意义。


  3. 渲染器（Renderer）

         渲染器的作用是将相机能够观察到的物体绘制(渲染)出来，使我们可以在显示器中能够观察到图形元素

three.js各组件之间的关系

  1. 页面加载three.js

    
                    <script src="js/three.min.js"></script>

  2. 创建场景，相机，以及渲染器

    
                    //创建场景
                    var scene = new THREE.Scene();

                    //相机可见区域高宽比
                    var aspect = window.innerWidth / window.innerHeight;
                    //创建相机
                    var camera = new THREE.PerspectiveCamera( 75, aspect, 0.1, 1000 );
                    //设置相机位置
                    camera.position.z = 5;

                    //创建渲染器
                    var renderer = new THREE.WebGLRenderer();
                    //设置渲染区域的高和宽
                    renderer.setSize( window.innerWidth, window.innerHeight );
                    //将渲染器添加到浏览器指定区域
                    document.body.appendChild( renderer.domElement );

  1. 添加正方体

    
                    //创建一个正方体几何形状
                    var geometry = new THREE.BoxGeometry(1, 1, 1);
                    //创建法向量网孔材料
                    var material = new THREE.MeshNormalMaterial();
                    //将几何形状和材料通过网格模型合并生成最终的正方体
                    var cube = new THREE.Mesh(geometry, material);
                    //将正方体添加到场景中
                    scene.add(cube);

  2. 制作动画

    
                    var render = function () {
                        //旋转
                        cube.rotation.x += 0.1;
                        cube.rotation.y += 0.1;
                        //将相机和场景添加到渲染器，开始渲染
                        renderer.render(scene, camera);
                        //递归调用
                        requestAnimationFrame(render);
                    };
                    //开始动画
                    render();
